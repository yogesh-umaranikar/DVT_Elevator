# DVT Elevator Simulation System

A modular, extensible elevator control system built with C# and .NET 8, following **SOLID principles**, **Clean Architecture**, and equipped with unit tests using **NUnit**.

---

##  Features

-  Simulates one or more elevators operating in a multi-floor building
-  Follows **Single Responsibility**, **Open/Closed**, and other SOLID principles
-  Clean separation of concerns with abstractions for request handling and movement logic
-  Fully unit tested using **NUnit** and **Moq**
-  Designed for real-time simulation and extensibility
